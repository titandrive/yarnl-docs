# Bluetooth Controllers

Control Yarnl without putting down your hook. Most Bluetooth controllers work immediately once paired. If your controller sends standard keys (arrows, letters, etc.), Yarnl will detect them automatically.

---

## Setup

1. Pair your Bluetooth device with your phone, tablet, or computer
2. Open a pattern in Yarnl
3. Start pressing buttons

If your controller uses standard keyboard keys, it should work right away. If nothing happens, see [Media Controllers](#media-controllers) below.

---

## Media Controllers

Controllers designed for TikTok or media playback often send media keys (play, pause) instead of standard keys. If yours isn't detected, enable **Bluetooth/Media Remote** in **Settings** > **Keyboard Shortcuts**.

:::note
Due to browser limitations, you'll need to tap the page once before media keys will work.
:::

## Customizing Controls

All key bindings are configurable in [Keyboard Shortcuts](../settings/keyboard-shortcuts). Each action supports up to 3 alternate key bindings, so you can set up your controller however you like.
