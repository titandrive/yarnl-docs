# Keyboard Shortcuts

Customize keyboard shortcuts for the PDF pattern viewer.

---

## Options

| Setting | Description |
|---------|-------------|
| **Bluetooth/Media Remote** | Enable use of media keys (play/pause, next, previous) from a Bluetooth remote or media keyboard. Requires Yarnl to be the active media app on your device. |
| **Arrow Keys Scroll PDF** | When enabled, arrow keys scroll the PDF page. When disabled, arrow keys control counters and page navigation instead. |

---

## Shortcut Bindings

Each action supports up to 3 alternate key bindings. Click a key slot to record a new key.

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Increase Counter** | Up Arrow | Add one to the selected counter |
| **Decrease Counter** | Down Arrow | Subtract one from the selected counter |
| **Previous Page** | Left Arrow | Go back one page |
| **Next Page** | Right Arrow | Go forward one page |
| **Start/Stop Timer** | Space | Toggle the project timer on/off |
| **Next Counter** | Tab | Cycle to the next counter |
| **Zoom In** | + | Increase the PDF zoom level |
| **Zoom Out** | - | Decrease the PDF zoom level |
| **Exit Viewer** | Escape | Close the viewer and return to the library |

| Button | Description |
|--------|-------------|
| **Reset to Defaults** | Resets all shortcuts back to their default key bindings. |
