# Pattern Viewer

Where the magic happens. Track your progress while you craft.

---

## Row Counters

Keep track of where you are in a pattern.

<div className="steps-grid">

<div className="step-box">

### Create

Click **Add Counter** → name it (e.g., "Body", "Sleeves")

</div>

<div className="step-box">

### Use

**+** to increment, **-** to decrement, reset icon to start over

</div>

</div>

### Multiple Counters

Create as many as you need:

| Counter | Use Case |
|---------|----------|
| **Main** | Current row/round |
| **Body** | Body section |
| **Sleeves** | Each sleeve separately |
| **Repeat** | Pattern repeats within a row |

:::tip
All counter values save automatically — even if you close the app!
:::

---

## Timer

Track time spent on each project.

- Click the **timer icon** to start/pause
- Each pattern has its own timer
- Time accumulates across sessions
- Feeds into your overall stats
