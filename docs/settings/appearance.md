# Appearance

Customize the look and feel of your app.

---

## Theme

| Setting | Description |
|---------|-------------|
| **Color Theme** | Choose a color scheme for the app. Includes light and dark variants. |
| **Surprise Me** | Randomly selects a theme, font, and mascot. |
| **Auto Switch** | Automatically switch between light and dark mode. |
| **Auto Type** | How auto switch determines the mode. **System** follows your device's light/dark preference. **Scheduled** switches at specific times you set. Only visible when Auto Switch is enabled. |
| **Day Starts / Night Starts** | Set the time of day for light and dark mode to activate. Only visible when Auto Type is set to Scheduled. |

---

## Font

| Setting | Description |
|---------|-------------|
| **Font Family** | Choose your preferred font from a list of popular Google Fonts (JetBrains Mono, Inter, Roboto, Open Sans, Lato, Poppins, Nunito, Raleway, Source Sans Pro, Ubuntu, Fira Sans) or enter a custom Google Font name. |
| **Google Font Name** | Enter the exact name of any font from Google Fonts. Only visible when Font Family is set to Custom. |

---

## Header

| Setting | Description |
|---------|-------------|
| **Gradient Header** | Use gradient colors on the header bar instead of a solid color. |
| **Theme Toggle** | Show a day/night mode toggle button in the header for quick switching. |
| **Show Logo** | Display the mascot image in the header. |
| **Mascot** | Choose which mascot character appears in the header. Opens a visual picker. |
| **Theme Mascot** | Automatically use a mascot that matches your current theme. |
| **Mascot Action** | What happens when you click the mascot. **Go Home** returns to the main page. **Open Last Pattern** opens the most recently viewed pattern. |
| **Show Tagline** | Display subtitle text below the app name in the header. |
| **Tagline Text** | Customize the subtitle text. Only visible when Show Tagline is enabled. Default: "Your self-hosted crochet companion". |

---

## Mobile

| Setting | Description |
|---------|-------------|
| **Haptic Feedback** | Vibrate on button taps when using the app on a mobile device (Android only). |

---

## Library

| Setting | Description |
|---------|-------------|
| **Home Page** | The page shown first when you open Yarnl. Choose between **In Progress** and **Library**. |
| **Tab Counts** | Show the number of patterns on each tab header (e.g., "Library (12)"). |
| **Auto In-Progress on Timer** | Automatically mark a pattern as "In Progress" when you start its timer. |

---

## Badge Visibility

Controls which badges are shown on pattern cards in the library.

| Setting | Description |
|---------|-------------|
| **Type Badge** | Show PDF or MD label on cards to indicate the pattern type. |
| **Status Badge** | Show the current status (In Progress / Complete) on cards. |
| **Category Badge** | Show the assigned category name on cards. |
| **Star Badge** | Show a star icon on favorited cards. |

---

## Pattern Viewer

| Setting | Description |
|---------|-------------|
| **Keep Screen Awake** | Prevent your screen from sleeping while viewing a pattern. Works on Chrome, Edge, and Safari. |
| **Default Zoom** | Initial zoom level when opening a PDF pattern. Options: Fit Page, Fit Width, 25%â€“200%. |
| **Page Mode** | How pages are displayed. **Paginated** shows one page at a time. **Scroll** shows all pages in a continuous scroll. |
| **Page Buttons** | Show next/previous page buttons in scroll mode. Only visible when Page Mode is set to Scroll. |
| **Auto Timer** | Automatically start the timer when you open a pattern. |

| Button | Description |
|--------|-------------|
| **Reset to Defaults** | Resets all appearance settings back to their default values. |
