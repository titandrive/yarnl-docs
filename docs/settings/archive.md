# Archive

Manage archived patterns and projects. See the [Archive guide](../guide/archive) for details on how archiving works.

---

## Behavior

| Setting | Description |
|---------|-------------|
| **Delete instead of archive** | When enabled, the archive button is replaced with a delete button that permanently removes files instead of archiving them. Enabling this will permanently delete all currently archived patterns and projects. |

---

## Archive Settings

These settings are only visible when "Delete instead of archive" is disabled.

| Setting | Description |
|---------|-------------|
| **Auto Empty** | Automatically permanently delete archived items after a set number of days. |
| **Empty After** | Number of days before archived patterns and projects are permanently deleted. Default: 30. Range: 1–365. Only visible when Auto Empty is enabled. |

---

## Archived Patterns

Lists all archived patterns with the total count. Each item can be restored back to the library or permanently deleted. A **Delete All** button appears when archived patterns exist.

---

## Archived Projects

Lists all archived projects with the total count. Each item can be restored or permanently deleted. A **Delete All** button appears when archived projects exist.
