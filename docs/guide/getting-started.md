# Getting Started

Welcome to Yarnl! Here's how to get going.

---

## 1. Add Your First Pattern

<div className="steps-grid">

<div className="step-box">

### Upload PDF

1. Click **Add Pattern**
2. Select your PDF file(s)
3. Done!

</div>

<div className="step-box">

### Create Markdown

1. Click **Add Pattern** → **Create New**
2. Write your pattern
3. Save

</div>

</div>

---

## 2. Organize Your Library

- Assign a **category** (Amigurumi, Wearables, etc.)
- Add **hashtags** for filtering (`#beginner`, `#gift`)
- Mark **favorites** for quick access

---

## 3. Start Crocheting

1. Open a pattern from your library
2. Use the **row counter** to track progress
3. Start the **timer** to log your time

---

## 4. Make It Yours

Go to **Settings** to:
- Choose your **theme**
- Set up **keyboard shortcuts**
- Configure **backups**
