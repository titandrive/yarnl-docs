# Account

Manage your account settings.

---

## Profile

| Setting | Description |
|---------|-------------|
| **Logged in as** | Displays your current username and role. |
| **Change Username** | Set a new username. Only available if the admin has granted you permission to change your username. |

---

## Password

| Setting | Description |
|---------|-------------|
| **Change Password** | Update your password. Requires entering your current password and confirming the new one. Only available if the admin has granted you permission to change your password. |
| **Remove Password** | Remove your password entirely (useful if you only use SSO). Requires entering your current password to confirm. Only available if you have a password set. |

---

## Single Sign-On

Only visible if SSO (OIDC) is enabled by an admin.

| Setting | Description |
|---------|-------------|
| **Link SSO Account** | Connect your Yarnl account to an SSO provider for passwordless login. |
| **Unlink SSO** | Disconnect your SSO account. Only visible if an SSO account is linked. |

---

## Session

| Button | Description |
|--------|-------------|
| **Log Out** | End your current session and return to the login screen. |
