# Keyboard Shortcuts

Control Yarnl without putting down your hook.

---

## Default Shortcuts

| Action | Shortcut |
|--------|----------|
| Increment counter | `Space` or `↑` |
| Decrement counter | `↓` |
| Start/Stop timer | `T` |
| Next page | `→` or `Page Down` |
| Previous page | `←` or `Page Up` |
| Toggle fullscreen | `F` |

---

## Customize

**Settings** → **Keyboard Shortcuts**

1. Click on the action
2. Press your desired key
3. Saved automatically

## Mobile Gestures

| Gesture | Action |
|---------|--------|
| Swipe left | Next page |
| Swipe right | Previous page |
| Tap counter | Increment |
| Long press | Decrement |

---

## Troubleshooting

If a shortcut doesn't work:
1. Check for conflicts in Yarnl settings
2. Check browser shortcuts
3. Check OS-level shortcuts

**Reset to defaults:** Settings → Keyboard Shortcuts → Reset
