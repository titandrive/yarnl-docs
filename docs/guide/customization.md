# Customization

Make Yarnl yours.

---

## Themes

Go to **Settings** → **Appearance**

| Option | Description |
|--------|-------------|
| **Light** | Bright and clean |
| **Dark** | Easy on the eyes |
| **Auto** | Follows your system |

---

## Fonts

Choose what's comfortable for reading patterns:

- **Serif** — Classic, easy to read
- **Sans-serif** — Modern and clean
- **Monospace** — Great for aligned columns

---

## Mascots

Fun characters that appear throughout the interface. Pick your favorite or disable them entirely.
