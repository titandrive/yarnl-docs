# Getting Started

Welcome to Yarnl! This guide will help you get up and running quickly.

## Installation

:::note
Installation instructions will depend on your deployment method. Check the main installation guide for Docker, manual setup, or other options.
:::

## First Steps

Once Yarnl is running, here's how to get started:

### 1. Add Your First Pattern

You can add patterns in two ways:

**Upload a PDF:**
1. Click the **Add Pattern** button
2. Select your PDF file(s)
3. Yarnl will import them into your library

**Create a Markdown pattern:**
1. Click **Add Pattern** → **Create New**
2. Write your pattern using Markdown
3. Save to add it to your library

### 2. Organize Your Library

Once you have patterns imported:

1. Click on a pattern to open its details
2. Assign a **category** (Amigurumi, Wearables, etc.)
3. Add **hashtags** for easy filtering (`#beginner`, `#gift`, etc.)
4. Mark patterns as **favorites** for quick access

### 3. Start a Project

Ready to start crocheting?

1. Open a pattern from your library
2. The **Pattern Viewer** opens with your PDF or Markdown content
3. Use the **row counter** to track your progress
4. Start the **timer** to log your crafting time

### 4. Customize Your Experience

Make Yarnl feel like home:

1. Go to **Settings**
2. Choose your **theme** (dark/light/auto)
3. Set up **keyboard shortcuts** to your liking
4. Configure **backups** to protect your data
