Backups & Data

Never lose your progress.

What's Saved​

DataDescription
PatternsPDFs and Markdown
CountersAll values for all patterns
TimerTime spent on each project
PDF positionWhere you left off
SettingsAll customizations
LibraryCategories, hashtags, favorites

Creating Backups​

Manual​

Settings → Backups → Create Backup

Good before updates or migrations.

Scheduled​

Enable in Settings. Runs daily/weekly automatically.

Restoring​

  1. Settings → Backups
  2. Click Restore from Backup
  3. Select your backup file
  4. Confirm

Everything comes back exactly as it was.

Storage Options​

LocationUse Case
LocalSame server as Yarnl
External driveRedundancy
Network/NASShared storage
Cloud syncOff-site backup
tip

Store backups somewhere different from Yarnl itself.

Pushover Notifications​

Get notified when backups complete or fail.

  1. Create account at pushover.net
  2. Get your User Key
  3. Enter in Settings → Backups → Notifications

Best Practices​

  1. Enable scheduled backups
  2. Keep multiple copies
  3. Store off-site
  4. Test restores periodically
  5. Backup before updates
